%
%   For a UNH Thesis, the margins are 1.5" on the left, 1" at the bottom
%   and right.  The first page of a chapter or major section has a top
%   margin of 1.5".  On all other pages, the top margin is 1".
%
%   The parameters for the UNHTHESIS document style are set to meet the
%   above requirements.  However, exact placement of the text on the page
%   can vary from printer to printer, or even from week to week on the same
%   printer.  To compensate for this, two parameters can be used to shift
%   the entire text 'box' left to right and top to bottom:
%
%
%                                       |<-->| \hoffset
%                                       |    |
%                                       |    |
%                 +--------------------------+ --------
%                 |                          |       ^
%                 |                          |       |  \voffset
%                 |                          |       v
%                 |    +----------------+    | --------
%                 |    |              xx|    |
%                 |    |                |    |
%                 |    |  xx xxxxxxx xxx|    |
%                 |    |xxxx xx x xxx xx|    |
%                 |    |xxx xxx x xxxx x|    |
%                 |    |x xxx xxx x xxxx|    |
%                 |    |xxx xxx xxxx xxx|    |
%                 |    |x xx xxxx.      |    |
%                 |    +----------------+    |
%                 |                          |
%                 |                          |
%                 +--------------------------+
%
%   Negative values for \hoffset shift the text left on the page.
%   Negative values for \voffset shift the text up on the page.
%
%
